EVA 2: Panel de Gestión de Usuarios

Que le pediste a la IA: Para la realización de la presenta página, 
lo que se le solicitó a la IA fue que me generara el código de una pagina web para un panel de gestión de usuario, 
la cual incluyera las indicaciones técnicas solicitadas como por ejemplo los campos de  ingreso de datos como nombre de usuario, 
edad, clasificación entre usuario o administrador y usuario, las validaciones como por ejemplo que no queden vacíos los campos, 
los rango de ingreso de edad, la estructura de la funciones, etc.

Que mejoras se aplicaron al código: Dentro de las mejoras aplicadas, se agregaron datos de usuarios de muestra distintos a los propuestos por la IA, 
se organizó la estructura del proyecto en 3 archivos distinto los cuales contienen el HTML, 
la java script y el CSS, se verificó el nombre de las funciones para que mantengan una estructura coherente e intuitiva con el proyecto, 
y se realiza una breve modificación en el Índex para mantener la integridad de bootstrap 5. Por último para terminar, 
se revisa la estructura total del código de todos los archivos con el fin de que el proyecto cumpla con las características solicitadas, 
en lo cual se concluye que si se encuentra según los parámetros solicitados.
